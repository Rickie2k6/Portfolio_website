Papers

OUNLP at TSAR 2025: Multi-Round Text Simplification via Code Generation

Code

Fourth Workshop on Text Simplification, Accessibility and Readability (TSAR 2025)

We present the OUNLP system submitted to the TSAR-2025 shared task on readability-controlled text simplification. Our analysis reveals that simplification performance is strongly influenced by the gap between source and target CEFR levels, motivating a multi-round simplification strategy.

  • Designed two multi-round simplification approaches: MRS-Rule and MRS-Joint, generated using GPT-4o.
  • Showed that larger CEFR level gaps benefit from iterative, multi-round rewriting.
  • Ranked 7th out of 20 teams in the TSAR-2025 Shared Task, with follow-up improvements demonstrating further gains from LLM-initialized candidates.
